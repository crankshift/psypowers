# session-structuring

Session structure is the scaffolding that holds therapeutic work together. Without it, sessions drift, run over time, leave critical content unprocessed, and fail to build momentum between meetings. This skill provides frameworks for organizing sessions by modality and session type, with specific time allocations and transition guidance.

## Generic session structure (60 minutes)

This is the default template. Modality-specific structures below modify this base.

| Phase | Duration | Purpose | Key actions |
|---|---|---|---|
| Opening / check-in | 5 min | Establish connection, assess current state | Brief measure (ORS), mood check, "How has the week been?" |
| Homework / between-session review | 10 min | Bridge from last session | Review assigned tasks, discuss what worked/didn't, reinforce engagement |
| Agenda setting | 5 min | Collaborative focus | "What's most important to work on today?" Prioritise 1–2 items |
| Main intervention | 25 min | Core therapeutic work | Modality-specific techniques, skill teaching, processing |
| Session summary | 5 min | Consolidate learning | Therapist and client co-summarise key takeaways |
| Homework / between-session assignment | 5 min | Extend work beyond session | Assign specific, achievable tasks linked to session content |
| Closing / check-out | 5 min | Safe transition out | SRS administration, grounding if needed, confirm next appointment |

**For 50-minute sessions:** Compress main intervention to 20 min. Combine session summary with homework assignment into a single 5-min block. Alternatively, if homework review is minimal, shorten it to 5 min and preserve 25 min for intervention.

**For 45-minute sessions:** Reduce opening to 3 min, homework review to 5 min, intervention to 20 min, combined summary/homework/closing to 7 min. This is tight — consider extending to 50 min if clinically possible.

## CBT session structure

Cognitive Behavioural Therapy sessions follow a predictable, agenda-driven structure that models organised thinking and gives the client a sense of mastery.

| Phase | Duration (60 min) | Content |
|---|---|---|
| Mood check | 5 min | Administer PHQ-9/GAD-7 or brief check; compare to last session; note trajectory |
| Bridge from previous session | 5 min | "What stood out from last session?" Check retention of key concepts |
| Homework review | 10 min | Review thought records, behavioural experiments, exposure logs. Troubleshoot barriers. Reinforce effort regardless of outcome |
| Agenda setting | 5 min | Collaboratively set 1–2 agenda items. Therapist may suggest items from formulation. Client has input and veto |
| Cognitive / behavioural work | 25 min | Identify automatic thoughts → examine evidence → develop balanced alternatives. OR plan/review behavioural activation schedule, exposure hierarchy step, problem-solving exercise |
| Session summary + homework | 5 min | "What are you taking away from today?" Assign homework directly linked to session work. Write it down — both therapist and client should have a copy |
| Feedback | 5 min | SRS or informal check: "How did today's session feel?" |

**CBT-specific principles:**
- Socratic questioning over direct instruction — guide discovery, do not lecture.
- Homework is non-negotiable in CBT. If a client repeatedly does not complete homework, this IS the agenda item — explore barriers collaboratively.
- Use the cognitive model (situation → thought → emotion → behaviour) as the organising framework for each session's intervention.
- Thought records introduced early; behavioural experiments follow once cognitive restructuring basics are established.
- Psychoeducation is threaded throughout, not confined to a single session.

## DBT session structure (individual)

Dialectical Behaviour Therapy individual sessions follow a target hierarchy, not a flexible agenda. The therapist directs focus based on which target is most pressing.

| Phase | Duration (60 min) | Content |
|---|---|---|
| Mindfulness exercise | 5 min | Brief grounding or mindfulness practice to open. Sets dialectical frame |
| Diary card review | 10 min | Review the week's diary card: urges, emotions, skills used, target behaviours. Identify highest-priority target |
| Target hierarchy work | 30 min | Address targets in order: (1) life-threatening behaviours, (2) therapy-interfering behaviours, (3) quality-of-life-interfering behaviours, (4) skills generalisation. Use behavioural chain analysis for problem behaviours. Use solution analysis to generate alternatives |
| Skills coaching / practice | 10 min | Teach or rehearse relevant skill module (distress tolerance, emotion regulation, interpersonal effectiveness, mindfulness). Connect skill to the session's target behaviour |
| Wind-down | 5 min | Summary, validation, bridge to next session |

**DBT target hierarchy (always address in this order):**

| Priority | Target | Examples |
|---|---|---|
| 1 | Life-threatening behaviours | Suicidal behaviour, self-harm, homicidal behaviour |
| 2 | Therapy-interfering behaviours | Missing sessions, not completing diary card, crossing boundaries, therapist burnout behaviours |
| 3 | Quality-of-life-interfering behaviours | Substance abuse, eating disorder behaviours, housing instability, relationship crises |
| 4 | Skills generalisation | Increasing use of DBT skills in daily life |

**DBT-specific principles:**
- The diary card is the session's anchor. Without it, the therapist cannot identify targets. If the client did not complete the diary card, that IS a therapy-interfering behaviour (Priority 2).
- Chain analysis: activating event → vulnerability factors → links (thoughts, feelings, behaviours) → problem behaviour → consequences. Solution analysis identifies where to insert different responses.
- Balance validation and change strategies — this is the core dialectic.
- Phone coaching between sessions is part of standard DBT — brief, focused on skill application in crisis moments.

## DBT skills group structure

| Phase | Duration (120 min) | Content |
|---|---|---|
| Opening mindfulness | 10 min | Guided mindfulness exercise — rotating leadership among group members |
| Homework review | 30 min | Each member shares homework practice. Group provides feedback. Therapist reinforces skill use |
| Break | 10 min | — |
| New skill teaching | 30 min | Didactic presentation of new skill. Use handouts, whiteboard, examples |
| Practice exercise | 25 min | Role-play, worksheet, or experiential exercise applying the new skill |
| Wind-down | 15 min | Observations from practice, troubleshooting, assign homework for the week |

## ACT session structure

Acceptance and Commitment Therapy sessions are experiential and values-driven. Less structured than CBT but not unstructured.

| Phase | Duration (60 min) | Content |
|---|---|---|
| Present-moment check-in | 5 min | Mindfulness exercise OR brief defusion exercise to establish psychological flexibility stance |
| Between-session review | 10 min | Review values-based actions taken during the week. What was practised? What was noticed? |
| Experiential exercise | 20 min | Metaphor-based or experiential work targeting one of the six core processes: acceptance, defusion, present moment, self-as-context, values, committed action. The exercise should be chosen based on where the client is "stuck" in the hexaflex |
| Values exploration | 10 min | Revisit or deepen values clarification. Connect the session's work to what matters to the client. Distinguish values from goals |
| Committed action planning | 10 min | Identify specific, concrete, values-aligned actions for the coming week. Make them SMART where possible. Anticipate barriers and willingness to experience them |
| Closing | 5 min | Brief summary, SRS, grounding |

**ACT hexaflex — six core processes (choose session focus):**

| Process | Opposite (pathology) | Session focus when targeting this |
|---|---|---|
| Acceptance | Experiential avoidance | Willingness exercises, "passengers on the bus" |
| Defusion | Cognitive fusion | "I'm having the thought that...", word repetition, leaves on a stream |
| Present moment | Dominance of past/future | Mindfulness, five senses, here-and-now focus |
| Self-as-context | Attachment to conceptualised self | Observer exercises, "you are not your thoughts" |
| Values | Lack of direction / values confusion | Values card sort, eulogy exercise, compass metaphor |
| Committed action | Inaction / impulsivity / avoidance | SMART goals, graduated exposure, behavioural activation |

**ACT-specific principles:**
- The therapist models psychological flexibility — self-disclose appropriately, be present, notice your own fusion.
- Do not argue with the content of thoughts — ACT is about changing the relationship to thoughts, not the thoughts themselves.
- Metaphors are the primary teaching tool (quicksand, tug-of-war with a monster, unwelcome party guest).
- Values are directions, not destinations — they cannot be "achieved" and are never complete.

## EMDR session structure

Eye Movement Desensitisation and Reprocessing follows the 8-phase protocol. A single processing session (phases 3–7) typically requires 90 minutes.

| Phase | Duration (90 min) | Content |
|---|---|---|
| Check-in (Phase 1–2 review) | 10 min | How has the client been since last session? Any disturbance? Review container/safe place exercise practice |
| Target identification (Phase 3) | 10 min | Select target memory. Identify image, negative cognition (NC), positive cognition (PC), VOC (1–7), emotion, SUD (0–10), body location |
| Desensitisation (Phase 4) | 30–45 min | Bilateral stimulation (eye movements, taps, or tones) while holding target. Process until SUD = 0 or ecologically valid level. Follow associative chains. Return to target periodically |
| Installation (Phase 5) | 5–10 min | Pair target memory with positive cognition. Bilateral stimulation until VOC = 7 or ecologically valid level |
| Body scan (Phase 6) | 5 min | Scan body while holding target + PC. Process any residual tension with bilateral stimulation |
| Closure (Phase 7) | 10 min | Contain any unprocessed material (container exercise). Orient to present. Debrief. Prepare client for between-session processing. Provide grounding instructions |
| Re-evaluation (Phase 8) | Next session | Check target memory — has SUD remained low? Any new associations or disturbance? |

**EMDR-specific principles:**
- Phase 2 (preparation) may take 1–4 sessions before any processing begins — this is where resourcing, psychoeducation, and stabilisation happen. Do not skip this with complex trauma.
- Processing does not always complete in one session. If SUD has not reached 0 by session end, close the session safely (container exercise) and resume next session.
- "Looping" (same material cycling without change in SUD) may require cognitive interweave — a therapist-introduced statement to restart processing.
- 90-minute sessions are strongly recommended for EMDR. 50-minute sessions are often insufficient for processing and safe closure.

## Psychodynamic session structure

Psychodynamic therapy is intentionally less structured. The "structure" is the frame (time, frequency, boundaries) within which free exploration occurs.

| Phase | Approximate timing | Content |
|---|---|---|
| Opening | Client-led | Client begins where they choose. Therapist listens. No agenda is set by the therapist |
| Middle | Bulk of session | Therapist listens for themes: transference patterns, defences, unconscious conflicts, relational patterns. Interventions include clarification, confrontation (pointing out discrepancies), interpretation (connecting unconscious meaning). Attention to the therapeutic relationship as a live sample of relational patterns |
| Closing | Final 5–10 min | Therapist may offer a summary observation or interpretation. Note any themes that emerged. No formal homework, but may invite reflection. Flag any strong feelings that arose — "We touched on something important today" |

**Psychodynamic-specific principles:**
- Silence is a tool, not a problem to solve.
- The relationship IS the intervention — attending to what happens between therapist and client is primary.
- Resistance is information, not an obstacle.
- The therapist's countertransference is clinical data — track it, supervise it, use it.
- Sessions typically 45–50 minutes, 1–3 times per week. Higher frequency deepens the work.

## First session vs follow-up sessions

The first session (intake/initial assessment) differs significantly from ongoing treatment sessions.

| Component | First session | Follow-up sessions |
|---|---|---|
| Primary goal | Assessment, rapport, treatment framing | Therapeutic work |
| Rapport building | Extensive — foundational | Maintenance |
| Assessment | Full history: presenting problem, psychiatric history, medical, family, social, developmental, substance use, risk, strengths | Targeted: "What happened this week?" |
| Informed consent | Required — explain confidentiality, limits, treatment approach, fees, cancellation policy, emergency procedures | Reference as needed |
| Goal setting | Collaborative initial goals | Review/refine goals over time |
| Psychoeducation | Explain the therapy model, what to expect | Thread in as relevant |
| Measurement | Baseline measures (PHQ-9, GAD-7, etc.) | Repeated measures for tracking |
| Homework | Minimal or none (perhaps self-monitoring) | Routine and modality-specific |
| Duration | May run 75–90 min (extended intake) | Standard session length |

**First session checklist:**
- [ ] Informed consent signed
- [ ] Emergency contacts obtained
- [ ] Presenting problem and history taken
- [ ] Risk assessment completed (suicidality, self-harm, violence, abuse)
- [ ] Baseline measures administered
- [ ] Preliminary formulation shared (even if tentative)
- [ ] Initial treatment goals agreed
- [ ] Expectations and logistics clarified (frequency, duration, communication between sessions)
- [ ] Next appointment scheduled

## Group therapy session structure

| Phase | Duration (90 min) | Content |
|---|---|---|
| Opening round | 10 min | Each member briefly checks in — mood, salient event, one sentence on how they are arriving. Keeps it brief to prevent one member dominating |
| Skills teaching / theme introduction | 15 min | Didactic component: introduce concept, skill, or theme for the session. Use whiteboard, handout, or brief presentation |
| Practice exercise | 20 min | Structured activity: role-play, worksheet, small-group discussion, experiential exercise |
| Processing / group discussion | 30 min | Open discussion of the exercise and how it connects to members' lives. Facilitator manages airtime, draws in quieter members, manages dominant speakers |
| Closing round | 10 min | Each member shares one takeaway. Facilitator summarises themes. Assign between-session practice |
| Buffer | 5 min | Administrative, scheduling, informal connection |

**Group-specific principles:**
- Establish group norms in session 1: confidentiality, respect, one person speaks at a time, regular attendance, right to pass.
- Track speaking time — aim for equitable distribution across members.
- Co-facilitation is strongly recommended for groups — one facilitator leads content, one manages process.

## Managing session overruns

**When heavy content emerges in the final 10 minutes:**

1. **Acknowledge and validate.** "What you're sharing is really important, and I want to give it the attention it deserves."
2. **Name the time boundary.** "We have about 5 minutes left today."
3. **Contain.** "Let's find a way to hold this safely until next session. Can you imagine placing this in a container — a box, a safe — and closing it? You're not forgetting it; you're setting it aside for now."
4. **Plan.** "Let's make this our starting point next session."
5. **Ground.** Use a brief grounding exercise (5-4-3-2-1 senses, feet on the floor, deep breath).
6. **Safety check.** "Are you safe to leave today? Do you need anything before you go?"

**Prevention strategies:**
- Set the agenda collaboratively at the start — allocate time and stick to it.
- At the session midpoint, briefly check: "We have 25 minutes left — where do we want to focus?"
- Avoid introducing new topics after the 40-minute mark in a 60-minute session.
- If a client routinely brings up the hardest content at the end ("doorknob disclosures"), address the pattern directly: "I've noticed we sometimes get to the most important things right as we're ending. What do you think is happening there?"

## Ending sessions safely

**Standard closing sequence:**
1. **Summarise.** Ask the client to summarise first: "What are you taking away from today?" Supplement if needed.
2. **Assign between-session work.** Link it directly to the session's content. Make it specific and achievable.
3. **Transitional technique.** Give the client something to carry: a written summary, a coping card, a skill to practise, a question to reflect on.
4. **Grounding.** If heavy content was processed, use a brief grounding exercise before the client leaves. Especially important after EMDR, trauma-focused work, or emotionally intense sessions.
5. **Safety check.** If there was any suicidal ideation, self-harm discussion, or high distress: "On a scale of 0–10, how are you feeling right now? Are you safe to drive / get home? Do you have your safety plan?"
6. **Confirm logistics.** Next session date/time. How to reach therapist if needed between sessions.

**After particularly heavy sessions:**
- Consider a brief check-in call or message the next day (within boundary agreements).
- Front-load the next session with the client's response to the heavy session.
- Document the client's state at session end in your notes.

## Common mistakes

- **No agenda setting.** Without a shared agenda, sessions meander and critical content is missed. Even in psychodynamic work, the therapist holds an internal frame.
- **All talk, no action.** In CBT and DBT, the session should include active practice (thought records, role-plays, exposure) — not just discussion about problems. If every session is "venting," the structure has collapsed.
- **Skipping homework review.** If homework is not reviewed, clients learn it is optional. This undermines the between-session practice that drives most therapeutic change.
- **Running over time.** Chronic overruns erode boundaries, model poor self-regulation, and can indicate therapist difficulty with limit-setting — which may parallel the client's issues.
- **Introducing heavy content late.** The therapist is responsible for time management. Do not open trauma processing with 15 minutes left in a 50-minute session.
- **Rigid adherence to structure at the expense of the client.** Structure is a guide, not a cage. If a client arrives in crisis, the agenda is the crisis. If a client is sobbing, you do not interrupt to review homework.
- **Neglecting the therapeutic relationship.** In modality-specific structures (especially CBT), it is easy to become a technician. Alliance is the strongest predictor of outcome across all modalities — monitor it (SRS) and attend to it.
- **No measurement.** Every session should include at least one brief measure (ORS at start, SRS at end). Without data, you are guessing about progress.
